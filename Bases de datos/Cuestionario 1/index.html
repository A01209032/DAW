¿Qué es entonces una base de datos - DB?

Una base de datos es una coleccion de archivos interrelacionados y facil de accesar.

¿En qué casos es conveniente usar bases de datos?

Cuando tienes un volumen muy grande de registros que manejar

¿Qué es un sistema de gestión de base de datos - DBMS

Una base de datos y un programa para accederla

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Crear y organizar la base de datos

Insercion: agrega algo a la base de datos
Actualizacion: actualiza la base de datos
Borrar: borra algo en la base de datos
Facilita y simplifica el acceso a los datos
Brinda seguridad, integridad y confidencialidad a los datos dentro de la base datos

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

Un ejemplo de un proyecto en el que pude haber usado una base de datos es el royecto que realice en mi primer semestre de la carrera para la materia de organizacion computacional.
Estre proyecto consistia en un terrarrio inteligente que se controlaba con diferentes factores como humedad y temperatura y aparte tenia una camara que podia ser accesada en red
La implementacion de una base de datos para todos los registros que median los sensores me hubiera ayudado a poder procesar de una mejor manera los datos que recibian, por ejemplo como variaba la temperatura y la humedad respecto a la hora del dia.
