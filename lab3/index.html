HTML tiene diferentes metodos que se pueden usar para recibir o mandar informacion a un servidor, los codigos de error se regresan si los metodos fallan

XML

Metalenguaje para mostrar cosas por internet [2]

XHTML

Lenguaje parecido a html pero mas estricto, es una aplicacion de XML [3]

HTML5

HTML5 es una nueva tecnologia mas facil de usar y con caracteristicas adicionales a xml y html, se usa para mostrar contenido en navegadores [4]

Javascript

Es un lenguaje orientado objetos para crear contenido interactivo en navegadores [5]

PHP

Es un lenguaje para hacer scripts para desarrollo WEB [6]

JSP

Tecnologia para crear dynamicamente paginas web basadas en html y xml [7]

Ajax

Es una tecnica para crear aplicaciones web usando XML, HTML, CSS y Javascript [8]
