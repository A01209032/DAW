¿Qué es ODBC y para qué es útil?

Es una interfaz que permite que se use lenuaje de sql standar en diferentes tipos de DBSM

Es util porque en caso de que se cambiara de DBSM no se tendrian que modificar los scripts

¿Qué es SQL Injection?

Cuando atraves de una form en una pagina web se injecta un script de sql y se corre en el servidor

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Checar que los datos que entran al servidor no tengan la sintaxis de un script de sql
