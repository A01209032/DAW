¿A qué se refiere la descentralización de servicios web?

A que estan distribuidos de una manera en que no hay dependencias entre ellos

¿Cómo puede implementarse un entorno con servicios web disponibles aún cuando falle un servidor?

Para implementarse de esta manera debe de haber mas sde un camino entre dos servidores, por lo tanto si falla un servidor no se rompen todos los caminos
